# Stock Analysis Report

This repository contains stock analysis reports that can be viewed directly in the browser.

## View the Report

To view the latest stock analysis report, simply open the HTML file directly in your browser or visit the GitHub Pages site.

## GitHub Pages

This repository is configured to serve content via GitHub Pages. The main report can be accessed at:
`https://[your-username].github.io/stock_analyze/2025-11-22-report.html`

## Files

- `2025-11-22-report.html` - The main stock analysis report
- `btc_data.py` - Python script for BTC data analysis
- `month_scipt.ipynb` - Jupyter notebook with monthly analysis
- `scripy.py` - Additional Python script
